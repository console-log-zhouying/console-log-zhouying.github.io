#### 一、介绍

- ##### 内存

  - 由可读写单元组成，表示一片可操作空间

- ##### 管理

  - 人为的去操作一片空间的申请、使用和释放

<!-- more -->

- ##### 内存管理

  - 开发者主动申请空间、使用空间、释放空间

- ##### 管理流程

  - 申请— 使用— 释放

    ```js
    // 申请
    let obj = {}
    
    // 使用
    obj.name = 'lg'
    
    // 释放
    obj = null
    ```
